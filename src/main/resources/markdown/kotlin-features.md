# Modern Kotlin Features

Kotlin is a cross-platform, statically typed, general-purpose programming language with type inference. It is designed to interoperate fully with Java.

## Why Kotlin?

1.  **Conciseness**: Drastically reduce the amount of boilerplate code.
2.  **Safe**: Avoid entire classes of errors such as null pointer exceptions.
3.  **Interoperable**: Use existing libraries for the JVM, Android, and the browser.

Our project leverages Kotlin's unique syntax to build a clean and modern Spring Boot application!
