# Spring Security Basics

Spring Security is a powerful and highly customizable authentication and access-control framework. It is the de-facto standard for securing Spring-based applications.

## Key Features

*   **Authentication**: Support for many authentication models (Form login, OAuth2, LDAP, etc.)
*   **Authorization**: Comprehensive and extensible support for authorization.
*   **Protection**: Protection against common exploits like CSRF, session fixation, clickjacking, and more.

In this blog, we've implemented a simple form-based login and used it to secure our comments!
